### Adrenalin-app

## How to run
- Install or update dependencies by running command `yarn add` for `yarn` or `npm install` when using `npm` on your terminal
- To run app, use command `yarn start` or `npm run start`

## Technologies Used
- react
- redux
- semantic-ui
- react-router
